We will do all the exercises on the Open Science Grid. If you do not have an account on the Open Science Grid, please sign up, either in advance of the course or first thing in the morning. It takes 5-10 minutes to complete the sign up process and about 30 minutes to get your account activated. Please make sure that you have the secured shell available on your laptop, before the start of your workshop.

Secured Shell and Secured Copy

ssh (secure shell) is an encrypted remote login tool. It lets you connect to remote resources, such as the OSG Connect login node and interact with the machine via UNIX shell. Similarly, scp (secure copy) is a tool that facilitates encrypted file transfer. We will use both to interact with OSG Connect.

Setup for Mac OS X and Linux

Most of them come with built in installation of ssh and scp, if not, install them.

Windows

Install PuTTY which is a free implementation of SSH for Windows.

Check your setup

To check your account on OSG using web shell:

  • Once you finished your Sign Up, the account activation may take about 30 minutes.
  • open WebShell
  • type in your username and password as prompted
  • if everthing is fine you will land on your home directory
